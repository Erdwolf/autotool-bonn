Graceful: Graceful labelings
Instanz G :: Graph a Ein Graph G
Beweis f :: FiniteMap a Integer ein graceful labeling
Eine Knotenbewertung f heißt graceful, wenn sie diese Bedingungen erfüllt:
  • Das kleinste Label ist die Zahl 0.
  • Das größte Label ist die Zahl e, wobei e die Anzahl der Kanten ist
  • Das Labeling f ist injektiv (keine Zahl kommt mehrfach vor)
  • Für die Multi-Menge M aller Kantendifferenzen { abs (f(x) - f(y)) : kante x y in kanten g } gilt: M = { 1 .. e }, d. h. jede mögliche Differenz kommt genau einmal vor.
Bemerkung: man vermutet, daß jeder Baum ein graceful labeling besitzt. Bis heute ist jedoch kein Beweis bekannt - ein Gegenbeispiel aber auch nicht.
