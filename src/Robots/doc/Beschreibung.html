Robots (Lunar Lockout)
Instanz mkKonfig [ Robot ] mit Robot { name :: String, position :: Position, ziel :: Maybe Position }
Beweis [(String, Richtung)] mit data Richtung = N | O | S | W
Das ist das Spiel Ricochet Robot von Alex Randolph (Verlag Hans im Glück), hier ohne Wände in der Version Lunar Lockout von Binary Arts.

Instanz ist eine Menge von Robotern. Beweis ist eine Zugfolge, durch die die gekennzeichneten Roboter (d. h. mit ziel = Just p) an ihr Ziel gelangen.
