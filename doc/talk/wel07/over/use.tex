Problem (Bsp: COL)
\begin{itemize}\itemsep 2pt
\item Instanz: ein Graph $G$ und eine Zahl $k$
\item Lösung: eine $k$-Färbung von $G$
\end{itemize}

Ablauf mit \autotool\ :
\begin{itemize}\itemsep 2pt
\item Tutor konfiguriert Generator
\item Student betrachtet Aufgabe: \\
  \autotool\ erzeugt persönliche Instanz
\item Student gibt (vermutete) Lösung ein
\item \autotool\ verifiziert Lösung, \\
  gibt ausführlichen Bericht (sofort)
\end{itemize}
