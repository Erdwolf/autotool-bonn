\begin{slide}{Typen (mehrsortige Algebren)}

Gesucht ist ein Ausdruck vom Typ \verb|boolean|  

in der Signatur
\begin{verbatim}
char a;
String b;
static String c ( boolean x );
static Bar d ( String x , char y , String z );
static boolean e ( Bar x , Bar y );
\end{verbatim}
Lösung:
\begin{verbatim}
e (d (b, a, b), d (b, a, b))
\end{verbatim}

\end{slide}
