\begin{slide}{Codes: Hamming-distance}

Find a code (as set of words over L,R) with
\begin{verbatim}
Config { width = ( Fixed , 4 ) 
    , size = ( Atleast , 5 )
    , distance = ( Atleast , 2 ) 
    , optimize = Size }
\end{verbatim}
\bigskip
solution
\begin{verbatim}
[ [L,R,R,L], [R,L,L,R], [L,L,L,L]
, [R,R,R,R], [L,L,R,R] ]
\end{verbatim}
discuss: error detection, error correction,
triangle inequality, bounds

\end{slide}
