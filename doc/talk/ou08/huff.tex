\begin{slide}{Huffman-Codes}

find an optimal binary prefix-free code for the frequencies
\begin{verbatim}
[ ( 'a' , 11 ) , ( 'b' , 47 ) , ( 'c' , 6 ) 
, ( 'd' , 20 ) , ( 'e' , 30 ) , ( 'f' , 31 ) ]
\end{verbatim}
    
\bigskip
form of solution
\begin{verbatim}
Code [ ( 'a' , [ R ] ) , ( 'b' , [ L , R ] ) 
    , ...
    , ( 'f' , [ L , L , L , L , L , R ] ) ]
\end{verbatim}

\end{slide}
