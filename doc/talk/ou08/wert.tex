\begin{slide}{Wertung von Aufgaben}

Aufgaben markieren als
\begin{itemize}
\item Demo (zu leicht, illustrieren Prinzip)
\item Mandatory (Pflicht, 
  Einsenden einer korrekten Lösung innerhalb der Deadline genügt)
\item Optional (zu schwer, zum Basteln, Highscore)
\end{itemize}

Student kann auch außerhalb der Deadlines 
\begin{itemize}
\item
  Aufgaben bearbeiten (wird korrigiert, nicht gespeichert)
\item
  vorige Bewertung ansehen
\end{itemize}
nützlich z. B. zur Prüfungsvorbereitung

\end{slide}
