\begin{slide}{Installation, Nutzung}

separate Installation benötigt
\begin{itemize}
\item
  halbwegs schnellen Rechner mit GNU/Linux, Apache Webserver,
  GHC-Compiler, MySQL-Datenbankserver (\dots \texttt{theo1})
\item
  dafür Administrator, der System und Datenbank einrichtet
  und im Betrieb bei Bedarf Patches einspielt und kompiliert
\end{itemize}

einfacher ist wohl die Benutzung eines gemeinsamen 
\autotool-Servers durch mehrere Einrichtungen (Uni und HTWK)
