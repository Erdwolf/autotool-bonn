\begin{slide}{Typische Anwendung}

Problem (Bsp: 3SAT)
\begin{itemize}
\item Instanz: eine Formel in 3-KNF
\item Lösung: eine erfüllende Belegung
\end{itemize}

Ablauf mit \autotool\ :
\begin{itemize}
\item Tutor konfiguriert Generator
\item \autotool\ würfelt Instanz
  (für jeden Studenten eine andere)
\item Student gibt (vermutete) Lösung ein
\item \autotool\ verifiziert Lösung,
  gibt ausführlichen Bericht (sofort)
\end{itemize}

\end{slide}
